# RS Energy — Energia Solar por Assinatura

Site institucional da RS Energy, empresa de energia solar renovável por assinatura.

## Tecnologias

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Lucide React (ícones)

## Estrutura de Páginas

- `/` — Página principal (landing page)
- `/privacy-policy` — Política de Privacidade
- `/terms-of-use` — Termos de Uso

## Componentes Principais

- `Navbar` — Navegação principal
- `Hero` — Seção principal com CTA
- `StatsBar` — Barra de estatísticas animada
- `HowItWorks` — Como funciona (4 passos)
- `WhoIsFor` — Para quem é o serviço
- `Benefits` — Benefícios com hover
- `Testimonials` — Depoimentos de clientes
- `Farms` — Carrossel de fazendas solares
- `PyaeSection` — Seção PYA'E mobilidade elétrica
- `CtaBanner` — Banner de chamada para ação
- `Contact` — Formulário de contato
- `Footer` — Rodapé com links e redes sociais

## Como rodar localmente

```bash
npm install
npm run dev
```

Acesse http://localhost:3000

## Entrega

Projeto entregue como arquivo ZIP.

Para rodar localmente:

1. Extrair o ZIP
2. Abrir a pasta no terminal
3. Instalar dependências: npm install
4. Rodar o servidor: npm run dev
5. Acessar: http://localhost:3000
